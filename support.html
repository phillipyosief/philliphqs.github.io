  YouTube-Converter



First check in the "Task Manager" if YouTube-Converter.exe is running. If not restart/start YouTube-Converter. 
If it's running check the Firewall options in Windows and allow it for YouTube-Converter otherwise it cannot interact with the Browser Addon.


The Server runs on your PC because from a Addon itself it's not possible to install YouTube-Videos without a Server.
The App replace the server side. 
So you download directly your YouTube-Videos from YouTube not like other YouTube Converter where you download the videos on there slow servers


YouTube-Converter needs the permission to interact with the Browser Addon.


It can happen that your Antivirus or even Windows Defender claims that this App is malicious. This happens because this App doesn't have an certificate yet (cause a certificate can be extremly expensive). The source code is free available if you still think this is a virus check the code and compile it yourself here is a small guide


This App doesn't have an updater after a new release your Browser Addon will automatically get updated (except you disabled it from your Browser settings) to update YouTube-Converter you need to install the new setup file which you find here
